Technical Insights: How Compression Works

Compression is a mathematical process of removing redundant or unnecessary data from an image file. There are two primary types:

1. Lossless Compression

This method shrinks the file size without removing any pixel data. It works by identifying patterns in the data and rewriting them more efficiently. While lossless compression is great for quality, it rarely achieves the massive size reduction needed for government forms.

2. Lossy Compression (Our Strategy)

Lossy compression intelligently discards information that the human eye cannot easily perceive. Our tool utilizes the DCT (Discrete Cosine Transform) algorithm via the browser's Canvas API. It focuses on reducing "Chroma" (color data) more than "Luma" (brightness data), which allows the image to look sharp while the file size drops by up to 90%.
