How to Create Blank Messages & Spoilers on WhatsApp

WhatsApp generally prevents you from sending empty messages. If you just type spaces, the "Send" button stays disabled. Our tool solves this by using the Braille Pattern Blank (U+2800) character, which is invisible but treated as valid text by WhatsApp.

New Feature: Add Text (Spoilers)

We have added a text box so you can now combine invisible space with visible text. This is perfect for:

  • Spoilers: Type "Warning: Spoiler!" then add 50 lines of blank space below it. The recipient will see the warning but has to tap "Read More" or scroll down to see the rest.
  • Hidden Pranks: Send a blank message that ends with "Gotcha!" at the very bottom.
  • Pure Blank: Simply leave the text box empty to send a classic ghost message.

Instructions

  1. Type Message: Enter any text you want visible (optional).
  2. Set Position: Choose if the blank space goes Before your text (Top) or After your text (Bottom).
  3. Count: Set how many blank lines you want. 10 is good for spacing; 500 triggers the "Read More" link.
  4. Send: Click "Send to WhatsApp" to open the app immediately.
