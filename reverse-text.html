What is a Reverse Text Tool?

A reverse text tool is a software utility that takes an input string and rearranges the sequence of its components. While it sounds simple, there are different ways to "reverse" language. You might want to reverse characters (turning "Hello" into "olleH"), or you might want to reverse words within a sentence (turning "Hello World" into "World Hello").
