How to Use the Four Swap Modes

1. Swap Adjacent Pair

This mode takes every two characters and flips their positions.

  • Logic: Character 1 swaps with Character 2. Character 3 swaps with Character 4.
  • Example: "Hello" becomes "eHll o".
  • Use Case: Simple cryptography or creating "alien" languages for RPGs and fiction writing. It makes text unreadable at a glance but easily decipherable if you know the pattern.

2. Scramble (Typoglycemia Mode)

This is the most popular feature. It keeps the first and last letter of every word in their correct positions but shuffles all the letters in the middle randomly.

  • Logic: "Problem" → P (fixed) + roble (shuffled) + m (fixed) → "Porlbem".
  • Use Case: Testing the cognitive theory that humans read words as a whole pattern, not letter-by-letter. You can send emails to friends that look messy but are surprisingly readable.

3. Specific Character Replace

This is a targeted swap. You tell the tool exactly which letter to find and what to swap it with.

  • Logic: Find "a", replace with "@".
  • Example: "Password" becomes "P@ssword".
  • Use Case: Leetspeak (1337) generation, creating secure passwords, or sanitizing text by masking specific characters.

4. Invert Case

This mode flips the capitalization of every letter.

  • Logic: A → a, b → B.
  • Example: "Code" becomes "cODE".
  • Use Case: Fixing text that was accidentally typed with CAPS LOCK on.

What is Typoglycemia?

Definition: Typoglycemia is a portmanteau of "typo" and "hypoglycemia." It refers to a viral internet meme/theory suggesting that as long as the first and last letters of a word are correct, your brain can read the scrambled mess in between.

Try it yourself using the "Scramble" mode above! Paste this paragraph in:

"According to a researcher at Cambridge University, it doesn't matter in what order the letters in a word are, the only important thing is that the first and last letter be at the right place."

The result will look chaotic, but you will find you can read it perfectly at normal speed. This tool is the easiest way to generate text to test this phenomenon.

Why Use a Client-Side Text Swapper?

Privacy and Security

When you are manipulating text—especially passwords or personal messages—you don't want that data sent to a server. This tool runs 100% in your browser. The scrambling logic is handled by JavaScript on your device. No data leaves your computer.

Speed

Server-side tools require a round trip (upload, process, download). This tool is instant. As you type, the letters swap in real-time.

Real-World Applications

1. Obfuscating Email Addresses

If you post your email address publicly, bots will scrape it. By using the "Specific Character Replace" mode to swap `.` with ` [dot] ` and `@` with ` [at] `, you make it readable for humans but broken for bots.

2. Creating Puzzles and Games

Teachers and game masters use the "Adjacent Swap" or "Scramble" modes to create clues. Players have to "descramble" the text to reveal the hidden message.

3. Cleaning Data

If you downloaded a dataset where a specific character is corrupted (e.g., all "é" characters turned into "?"), use the Replace mode to swap "?" back to "é" instantly.

Frequently Asked Questions (FAQ)

Can I un-scramble text with this tool?

If you used "Adjacent Swap" or "Invert Case," running the tool a second time on the result will revert it back to the original (because the logic is symmetrical).
However, the "Scramble" mode is random. You cannot automatically un-scramble it because the original order is lost.

Does this work on other languages?

Yes. The tool supports Unicode, so it works with Spanish, French, Russian, Chinese, emojis, etc.

Is there a character limit?

No limit. You can paste entire books into the tool, and it will process them in seconds.
