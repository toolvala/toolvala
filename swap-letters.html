What is the Swap Letters Tool?

Text formatting is usually rigid, but sometimes you need to break those rules to obfuscate data, test reading theories, or create engaging puzzles. The Swap Letters Tool is a fast, entirely browser-based utility designed to structurally manipulate characters within strings instantly.

Understanding the Four Modes

1. Swap Adjacent Characters

This mode flips the positions of every pair of characters. It’s an easy form of text scrambling that makes content unreadable at a passing glance, but entirely reversible if run through the tool a second time.

Input: Hello World Output: eHll ooWlrd

2. Word Scramble (Typoglycemia)

Have you ever seen a paragraph where the letters are jumbled, but you can still read it perfectly? This tool creates that exact effect. It keeps the first and last letter of every word in place, but randomly shuffles all the letters in the middle. This is perfect for educational reading tests or sending fun, encrypted messages.

3. Find & Replace Characters

A classic utility that allows targeted replacement of specific letters or phrases. Whether you need to replace all vowels with numbers to generate a strong password (Leetspeak) or obfuscate an email address to prevent spam bots from scraping it, this mode handles it instantly.

Input: admin@example.com Find: @ Replace: [at] Output: admin[at]example.com

4. Invert Case

Accidentally left Caps Lock on while typing an entire paragraph? The Invert Case mode will swap every uppercase letter to lowercase and vice versa, fixing your text without needing to retype it.

Is my text secure?

Absolutely. This tool runs 100% locally in your web browser using JavaScript. Your text is never uploaded, saved, or sent to any external server. You can safely paste sensitive emails, passwords, or personal documents.
